# Encurtou

Sistema de encurtamento de URLs desenvolvido com Spring Boot.

## Sobre o Projeto

Encurtou é uma aplicação web (local) que permite encurtar URLs longas, facilitando o compartilhamento e gerenciamento de links. O sistema gera códigos únicos e curtos que redirecionam para as URLs originais. As URLs originais e os códigos gerados ficam armazenados em um banco de dados PostgreSQL, garantindo persistência e disponibilidade dos links enquanto a aplicação estiver em execução, ou em interações futuras contanto que o banco de dados esteja disponível.

## Tecnologias Utilizadas

- Java 17
- Spring Boot
- PostgreSQL
- Maven
- IntelliJ IDEA Community Edition

## Funcionalidades

- Encurtamento de URLs longas
- Redirecionamento automático para URL original
- Geração de códigos únicos para cada link
- API RESTful para integração

## Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- Java 17 ou superior
- Maven 3.6+
- Git
